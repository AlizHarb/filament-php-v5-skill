# Filament PHP v5 Skill

A Laravel Boost skill for building Server-Driven UIs with Filament PHP v5.

## Quick Install

```bash
php artisan boost:add-skill alizharb/filament-php-v5-skill
```

## What You Get

This skill adds AI-powered knowledge for Filament PHP v5 development:

- **v5 Architecture** - New schemas, tables, and action patterns
- **Real Examples** - 15+ code samples from Filament 5.3.2
- **Best Practices** - 21 actionable development rules
- **Anti-patterns** - 12 common mistakes with corrections
- **Generation Commands** - 17 command examples
- **Testing Guidance** - Pest integration patterns

## Coverage

- Resources, Pages, Widgets
- Forms and Tables
- Actions (record, toolbar, bulk)
- Relation Managers
- Deployment and Testing

## Requirements

- Laravel 11+
- Filament 5.3.2+
- PHP 8.3+
- Livewire 4+

## Compatible Agents

Claude Code • Cursor • Copilot • Gemini • Codeium • Windsurf • Roo • Cline • OpenCode
